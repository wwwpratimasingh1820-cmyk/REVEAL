Title: Reveal — AI Media Metadata Inspector

Short Description:
Reveal is a frontend-only web app that lets you upload images and videos to extract full metadata and check for AI-generated content. It provides a clean, premium UI, supports dark/light mode, and shows detailed technical information about your media while remaining privacy-first and fully local — no backend or APIs required.

Features:

Extract full metadata from images and videos (EXIF, XMP, IPTC, container, encoder, track info)

AI detection using heuristic analysis with confidence explanation

Light/Dark mode toggle with smooth transitions

Advanced/raw metadata view for power users

100% frontend, privacy-respecting, no backend needed

Tech Stack:

HTML, CSS, JavaScript (vanilla)

Works in modern browsers (desktop & mobile)

Use Case:
Ideal for photographers, content creators, developers, and anyone curious about the origin and technical details of media files.
